שלב ראשון - הטמעת הספריה בתוך הפרויקט

ראשית לחצי על הכפתור להורדת הזיפ.

את הזיפ עליך לחלץ. תקבלי תיקיה בשם "fontawesome". הכניסי את התיקיה לתוך הפרויקט שלך.

לאחר מכן עליך להטמיע קישור לקובץ CSS שנמצא בתוך התיקיה, ושם בתוך תיקיה בשם css. הקובץ נקרא all.min.css. לדוגמה:

<link rel="stylesheet" href="fontawesome/css/all.min.css"/>

שלב שני - בחירה והצגה של אייקון באתר

בשדה החיפוש הקלידי את האייקון המבוקש. באנגלית כמובן. לדוגמה smile. מיד יופיעו לך בעמוד כל האייקונים שקשורים למה שחיפשת.
כעת לחצי על האייקון הדרוש לך. בפופאפ שנפתח יופיעו 2 כפתורים:

  • בעת לחיצה על הכפתור "Copy CSS for before/after" יועתק לך קוד CSS שתוכלי לתת לbefore או after.
  • בעת לחיצה על הכפתור "Copy HTML tag" - תועתק תגית HTML של האייקון, כדי להטמיע במקום הדרוש בקובץ הHTML שלך.
